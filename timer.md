## Features include
- Intuitive swipe gestures
- Multiple timers
- Timers synced on all devices
- Track time between timers
- Full screen progress animations
- Minimal mode
- Easily edit timers
- Name timers
- Favorite timers
- Timer history
- Live Activities
- Dynamic Island
- Widgets
- Colors
- And more!


Inspired by the amazing [Gestimer](https://maddin.io/gestimer/) app on macOS.
